# Cars Dealership - Full Stack Web Application

## Project Overview
This project is a comprehensive full-stack web application for a national car retailer in the U.S. It allows users to view dealership branches, read reviews, and post their own feedback using a microservices architecture.

## Technologies Used
- **Frontend:** React, HTML5, CSS3, JavaScript.
- **Backend:** Django, Python.
- **Microservices:** Node.js, Express.js.
- **Database:** MongoDB (Reviews), SQLite (Cars/Dealers).
- **Deployment:** Docker, Kubernetes, IBM Cloud Code Engine.
- **CI/CD:** GitHub Actions.
- **AI Integration:** Sentiment Analysis for customer reviews.

## Key Features
- Responsive UI for dealership browsing.
- User authentication (Login/Logout/Register).
- Sentiment Analysis for dealer reviews.
- Continuous Integration and Deployment.
